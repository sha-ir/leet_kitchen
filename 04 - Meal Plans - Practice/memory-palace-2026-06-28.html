Running counter, reset on 0

Increment on a 1, reset to 0 on a 0, take the max every step.

LC 0645 · Set Mismatch

Seen-twice vs never-seen

Second sighting = duplicate; the 1..n label never placed = missing.
s = set()
for n in nums:
    if n in s: dup = n
    s.add(n)
miss = next(i for i in
       range(1,len(nums)+1) if i not in s)
return [dup, miss]

Sort + first-occurrence map

In sorted order, a value's earliest index equals the count of strictly-smaller values.
m = {}
for i,v in enumerate(sorted(nums)):
    m.setdefault(v, i)   # keep first
return [m[n] for n in nums]

LC 0636 · Exclusive Time

Call-stack, subtract the child

Push on start. On end: exec = t − start + 1 to the top; subtract it from the parent.
st=[]; res=[0]*n
for id,s,t in parsed(logs):
    if s=="start": st.append([id,t])
    else:
        x,y = st.pop()
        e = t-y+1; res[x]+=e
        if st: res[st[-1][0]] -= e
return res
