# Recognition Drills

Recognition drills train the moment before coding.

## Drill 1 — Title hidden

Read only the prompt and constraints. Fill:

```text
Ingredients:
Constraints allow:
Recipe guess:
Why this recipe:
Why not the nearest confusable recipe:
Core invariant:
```

## Drill 2 — Ingredient flashcards

Use [[Ingredient Glossary]]. Cover the recipe column and guess the recipe from the ingredient.

## Drill 3 — Confusable pairs

Pick one pair from [[Confusable Ingredients]] and write one example prompt for each side.

## Drill 4 — Template from memory

Open a blank note and write the core template for one recipe before checking [[Recipe Index]].
