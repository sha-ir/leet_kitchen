# Daily Consumption Plan

Do **3 problems per pattern**, not 30 random problems.

For each pattern:

1. **Easy** — learn the recipe.
2. **Medium** — apply it without heavy hints.
3. **Medium/hard** — stress-test recognition and edge cases.

After each problem, fill [[Template - Problem Study Card]].

## Daily session script

| Time box | Action | Output |
|---|---|---|
| Warm-up | Review one due card from [[Review Dashboard]] | One recalled invariant |
| Taste test | Read prompt + constraints | Ingredients list |
| Cook | Solve using recipe template | Working code or failure note |
| Plate | Fill study card | Problem note |
| Taste again | Compare with confusable recipe | One sentence: why this recipe |

## Finish line

Your goal is not “I solved this problem.”

Your goal is:

> **I can recognize this dish again.**
