# LeetCode Cookbook Obsidian Vault

Open this folder as an Obsidian vault. Start with `START HERE.md`.

This vault is centered on the food-native recognition loop:

1. Ingredients
2. Constraints
3. Recipe card
4. Problem study card
5. Review and recognition drills

Optional: install Dataview if you want the dashboard code blocks to render as live tables. The vault still works without it.
