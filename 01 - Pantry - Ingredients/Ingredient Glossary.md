# Ingredient Glossary

Ingredients are prompt clues. Recipes are patterns. The goal is to train fast recognition.

| Ingredient | What it usually smells like | First recipe to check |
|---|---|---|
| contiguous substring/subarray | A moving region of adjacent items | [[Sliding Window Stir-Fry]] or [[Prefix Sum Marinade]] |
| sorted input | Order can eliminate choices | [[Two Pointers Sandwich Squeeze]] or [[Binary Search Taste Test]] |
| sum equals k | Compare current total with a previous total | [[Prefix Sum Marinade]] |
| count/frequency/seen before | Need quick lookup or labels | [[Hash Map Pantry Labels]] |
| smallest valid answer | Monotonic yes/no predicate | [[Binary Search Taste Test]] |
| parentheses/nested/undo | Last unresolved item matters | [[Stack Pancake Stack]] |
| next greater/nearest taller | Dominated items can be popped | [[Monotonic Stack Ordered Pancake Tower]] |
| top k/kth | Need only best candidates | [[Heap Buffet Priority Line]] |
| shortest path/min moves | Spread by levels | [[BFS Soup Spreading Outward]] |
| connected cells/path search | Explore depth or components | [[DFS Skewer Going Deep]] |
| prerequisites/dependencies | Need valid order | [[Topological Sort Cooking Order]] |
| merge connected groups | Components combine over time | [[Union-Find Dough Merging]] |
| min/max/ways/choices | State depends on earlier states | [[Dynamic Programming Layer Cake]] |
| all combinations/permutations | Generate search tree | [[Backtracking Pizza Bar]] |
| overlapping ranges | Sort and merge sheets | [[Intervals Lasagna Sheets]] |
| local best/scheduling | Greedy choice might be safe | [[Greedy Chefs Instinct]] |
