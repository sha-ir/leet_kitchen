# Cookbook Home

The vault is organized like a kitchen:

| Kitchen area | Vault folder | Purpose |
|---|---|---|
| Pantry | [[Ingredient Glossary]] | Signals, constraints, keywords, and prompt clues. |
| Recipe cards | [[Recipe Index]] | Pattern notes with templates, invariants, variations, and mistakes. |
| Dishes | [[Problem Cards Index]] | One study card per solved problem. |
| Meal plans | [[Daily Consumption Plan]] | Daily and weekly practice loops. |
| Tasting room | [[Recognition Drills]] | Reviews, mistakes, invariants, and spaced repetition. |
| Dashboards | [[Pattern Dashboard]] | Dataview-ready tables and manual indexes. |
| Source | [[LeetCode Cookbook Food-Native Pattern Guide]] | The attached guide this vault is centered around. |

## Most important habit

Every problem note should contain this sentence before code:

> This looks like **___** because **___**.

That sentence is the bridge between ingredients and recipe.
